# NeuroSync - Medical Device Monitoring System

A modern web application for monitoring and analyzing tremor data from patients with Parkinson's disease.

## Features

- 🏥 Real-time patient monitoring
- 📊 Interactive data visualization
- 🔔 Alert system for critical events
- 👥 Patient management dashboard
- 📱 Responsive design for all devices

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Recharts
- Lucide React Icons

## Installation

1. Clone the repository:
\`\`\`bash
git clone https://github.com/YOUR_USERNAME/neurosync-medical-device.git
cd neurosync-medical-device
\`\`\`

2. Install dependencies:
\`\`\`bash
npm install
\`\`\`

3. Start development server:
\`\`\`bash
npm run dev
\`\`\`

4. Open browser to `http://localhost:5173`

## Build for Production

\`\`\`bash
npm run build
\`\`\`
